🚦 Civic Issue Reporting Tool

A simple and user-friendly web platform where citizens can report civic problems online and track complaint status in real time.

---

📌 Project Overview

The Civic Issue Reporting Tool helps users report public issues like road damage, garbage problems, water leakage, and more.
Users can upload images, submit complaints, and track issue status easily.

This project is built to improve communication between citizens and local authorities for faster issue resolution.

⚙️ How It Works

1️⃣ User Reports Issue

Users fill out the complaint form with issue details.

2️⃣ Admin Reviews Complaint

Admin checks and verifies the complaint.

3️⃣ Issue Gets Resolved

The issue status is updated after resolution.
